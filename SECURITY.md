# Security Policy

## Supported use

This project is intended for local paper-trading research.
Live-trading execution is intentionally outside the supported scope.

## Secrets

Do not commit:

- `.env` files;
- Alpaca API keys;
- X/Twitter bearer tokens;
- account exports;
- logs containing account or order details.

Rotate any credentials that may have been exposed.

## Reporting issues

If you publish this repository, add your preferred private contact method here for reporting security issues.
